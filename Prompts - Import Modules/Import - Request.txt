---- Import Request Instruction Start ----

To import the request package from a external module, you can use the following command:
```
moduleRequest "github.com/metadiv-go-tech/<module>/model/request"
```

For example, to import the entity package from the `mod_meta_accounting` module, you can use the following command:
```
accountingRequest "github.com/metadiv-go-tech/mod_meta_accounting/model/request"
```
