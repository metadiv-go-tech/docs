---- Import Entity Instruction Start ----

To import the entity package from a external module, you can use the following command:
```
moduleEntity "github.com/metadiv-go-tech/<module>/model/entity"
```

For example, to import the entity package from the `mod_meta_accounting` module, you can use the following command:
```
accountingEntity "github.com/metadiv-go-tech/mod_meta_accounting/model/entity"
```
